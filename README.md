# py-fitness

A Fitness Tracking Webapp.

Intent:

- Demonstrate automation of deployment and server provisioning
- Demonstrate mobile-first web development strategy
- Exemplify UX/UI design principles
- Integrate frontend javascript framework
- Create REST API to interact with
- Integrate authentication through other websites (Google, Facebook, Twitter, etc.)
- Provide practicality to help users track workouts
- Object character recognition and algorithms to parse grocery receipts and track food buying patterns
- Reviews of gyms based on Natural Language Processing of Reviews, Sentiment Analysis, Tagging of gyms based on common terms
